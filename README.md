# Golang Wallet System

## Description
Implementation of wallet system using Golang. This repository is a part of my self learning at my semester break.

## Frameworks
- HTTP framework : Fiber
- SQL builder : Goqu
- Database : PostgreSQL
- Cache : Redis
- Email : Mailtrap
- Payment Gateway : Midtrans
